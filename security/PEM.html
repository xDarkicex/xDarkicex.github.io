Privacy-enhanced Electronic Mail

Privacy-enhanced Electronic Mail Is the go to technology for sending keys, certificates and other data online. I have used a hybrid RSA-AES encryption to encrypt my key and send it out encrypted along with the data that was being sent in my case this was json data that contained commands to send to a remote server, these command where being actively ran on a live server so It was of the upmost importance that the connection was encrypted at not just the server and client but along the way as well and AES is the type of encryption to do exactly that.
