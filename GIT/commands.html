Basic intro commands for github

Github commands

  • git clone:

  • This command can clone a remote repo onto your desktop.

  • git init:

  • Inits a directory as a git directory.

  • git add .:

  • Git add, add your files to be commited.

  • git commit -m "message here"

  • This commits the staged files and adds a commit message.

  • git push:

  • git push pushes the commited files to what ever branch you are on.

  • git pull:

  • git pull pull's from what ever branch you are on into your local directory.
