Hyper Text Transfer Protocol Secure

Also known as Transport Layer Security(TSL) is a more secure method of data transfer on the web, sites that have http/s will have a green secure lock which lets the user know they are a much secure site that doesnt allow outside scripts or files. HTTP/S is considered much slower because to connect the browser will connect to HTTP and hit a perminate redirect status code 302, then it will connect to https on port 443. Though this is true many in the industry are attempting to not only speed up https but enforce its use across the web. One of the best new feature of https the ability for your server to push a ton of files at once instead of the 6 or so files the the old standard supported, another feature is the new push feature which allowes backend developers to build in http.Push commands that allow the developers tell the browser to grab files before the dom requires them to speed up site load times.
