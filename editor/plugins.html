Editor Plugins

To plugin or not to plugin that is the question.

Plugins

Plugin are almost essential for your editor, if you want to have any extended features such as syntax highlighting for langauages that it did not originally support. The best golang syntax highlighting is with a plugin that is developed for this editor which is the main reason I love it. you can get many other plugins though such as linters for things like javascript or typescript.
